Projects

  • MAΘ Timers
    Set of nine stopwatch-style timers designed for Mu Alpha Theta math honor society competitions. Officially used in competition since the 2014 National Convention in Orlando, FL!
  • Ten-Second Planetarium
    HTML5 canvas visualization of the solar system if one Earth year were scaled down to ten seconds. Uses plenty of astrophysics I learned at the Summer Science Program 2013.
  • Resistance Universe Toolkit
    Android app that simulates the 5-10 player board games The Resistance and The Resistance: Avalon. My goal was to make the game playable without a physical game board.
  • Dayta: data for days
    Gives information about dates and pairs of dates, with an interface modeled after the Windows native date selector. Developed a working knowledge of AngularJS 1.
  • GamesVault
    Ruby on Rails web app that aggregates data from gaming information websites to streamline game-buying. Created UI and wrote Ruby and JS to interact with backend. (Currently inactive.)
  • MAΘ S-score Calculator
    A Python script that scrapes data from the Florida Mu Alpha Theta math honor society's website to calculate performance statistics. Provides optional LaTeX output.
