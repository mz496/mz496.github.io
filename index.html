PROJECTS

  • MAΘ Timers
    A set of nine stopwatch-style timers, designed for Mu Alpha Theta high school math honor society competitions and the special timing and alerting involved. It's been used in official competitions since the 2014 National Convention in Orlando, FL!
  • Ten-Second Planetarium
    A HTML5 canvas visualization of solar system orbits and orbital speed using the benchmark that one Earth year equals ten seconds. Includes relevant background and plenty of math and astronomy I learned at the Summer Science Program 2013.
  • Subvert: Resistance Universe Toolkit
    Android app that simplifies the 5-10 player board games The Resistance and The Resistance: Avalon by implementing a large subset of the gameplay logic. The (attained) goal was to make the game playable without a physical game board.
  • Dayta: data for days
    Gives information about dates and pairs of dates, with an interface modeled after the Windows native date selector. Developed a working knowledge of AngularJS 1.
  • GamesVault
    Ruby on Rails web app that aggregates data from gaming information websites in order to streamline the game-buying process. Created UI and wrote Ruby and Ajax to interact with the database. (Website is currently down.)
  • MAΘ S-score Calculator
    A Python script that scrapes data from the Florida MAΘ national high school math honor society's website to calculate performance statistics. Python 2.7 requires the urllib, BeautifulSoup4, and numpy modules. Optional LaTeX output requires the pgfplots and fancyhdr packages.
