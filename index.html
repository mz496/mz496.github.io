Welcome to my Github page! I'm interested in web development, and I'm always seeking to expand my knowledge base of languages and frameworks.

Projects

  • MAΘ Timers

    A set of nine stopwatch-style timers, designed for Mu Alpha Theta competitions and the special timing and alerting involved. HTML, CSS, JS.

  • Ten-Second Planetarium

    A visualization of solar system orbits and orbital speed using the benchmark that one Earth year equals ten seconds. Includes an information panel with relevant background. HTML, CSS, JS.

  • MAΘ S-score Calculator

    A script that fetches data from the Florida MAΘ website, does some calculations, and outputs the sum of each student's (from a specified school) top three s-scores nicely. An s-score is a performance statistic that combines a student's statewide ranking and test score. Python requires the BeautifulSoup4 and numpy modules. Optional LaTeX output requires the pgfplots and fancyhdr packages.

  • Spherical Geometry Visualization

    A script that makes a manipulable 3D visualization of spherical coordinates. Includes some explanations about spherical coordinates and how to use. Python requires the VPython module.
